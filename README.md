# Advanced Accu Weather CU project

 Чтобы запустить приложение:

-> weather-app -> app.py

Перед этим

-> -weather-app -> .env
apikey=

 Добавить свой api ключ от AccuWeather



Чтобы запустить бота:

-> bot -> aiogram_run.py

Перед этим

-> bot -> .env
TOKEN=

Добавить token своего бота
